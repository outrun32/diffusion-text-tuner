| Boundary | Description |
|----------|-------------|
| synthetic dataset files -> inspector | Local generated images, masks, metadata, and optional OCR results are untrusted generated inputs until validated. |
| inspector -> contact sheet/report | Reports/contact sheets may expose prompt text and generated images. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-03-03-02 | Information disclosure | Contact sheets/reports | mitigate | Write only to explicit output paths and document non-committable generated image/report boundaries. |
| T-03-03-03 | Denial of service | Image scanning | mitigate | Support max samples/contact-sheet sample limits and avoid model/OCR loading in default path. |
