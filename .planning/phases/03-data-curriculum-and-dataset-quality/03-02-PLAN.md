<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| prompt JSONL -> validator | User/generated JSONL can be malformed, huge, duplicated, or contain private prompt text. |
| manifest/report -> filesystem | Reports write local provenance that may include prompt paths and counts. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Tampering | `validate_prompt_dataset` | mitigate | Validate JSON line-by-line, track source hashes, and include line-numbered errors. |
| T-03-02-02 | Information disclosure | Reports/manifests | mitigate | Do not echo full prompt text in aggregate reports except small duplicate examples; document private prompt caution. |
| T-03-02-03 | Denial of service | Source hashing | mitigate | Hash only explicitly safe small text inputs and reference large generated files by path. |
</threat_model>
