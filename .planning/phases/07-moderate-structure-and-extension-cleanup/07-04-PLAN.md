<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SynthTIGER config/template→subprocess | User-provided template/config/runner paths are passed to a local subprocess. |
| raw metadata→dataset indexes | Rendered metadata becomes masked-SFT and AnyWord dataset records. |
| image/mask artifacts→tensor files | Local generated images and masks are encoded into `.pt` artifacts during explicit GPU phases. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-04-01 | Tampering | `render_phase` | mitigate | Preserve explicit runner/template/config arguments and `subprocess.run(..., check=True)` so render failures stop the build. |
| T-07-04-02 | Information Disclosure | dataset indexes/prompts | mitigate | Keep generated indexes/prompts under caller-provided runtime output paths and document generated data as non-committable. |
| T-07-04-03 | Denial of Service | `bake_latents_phase`/`encode_text_phase` | mitigate | Preserve explicit `--bake-latents` and `--encode-text` gates so GPU/model work only runs when requested. |
</threat_model>
